Moodle - block_tableau_bord

See doc :  https://poleumdev.github.io/moodle-block_tableau_bord/ 

    Fonctionnement
    ==============

Ce block peut être utilisé uniquement dans la page "/my" de Moodle.
Il permet d'afficher la liste de cours dans lesquels l'utilisateur est inscrit.
Pour chacun de ces cours des indications sont affichées :
    - Des notifications permettant d'indiquer des nouveautés dans le cours ou des échéances à venir
    - Des indications concernant l'avancement dans le cours

Un système de drag and drop est également présent et permet à l'utilisateur d'organiser ses cours dans l'ordre qu'il souhaite.
L'utilisateur peut également modifier la disposition des cours et passer d'un affichage sous forme de liste à un affichage
 sous forme de grille.
